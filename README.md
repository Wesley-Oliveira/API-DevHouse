# DevHouse
    API developed to study NodeJS using the express framework.

# Routes and Methods
    # Houses
        POST, GET, DEL - /houses
        PUT - /houses/id
    # Dashboard
        GET - /dashboard
    # Sessions
        POST - /sessions
    # Reserves
        POST - /houses/id/reserve
        GET - /reserves
        DEL - /reserves/cancel
        
# Install dependencies
    yarn

# Execution after yarn configuration
    For backend execution: yarn dev
